Online Campus - Mobile App

The Online Campus Mobile App is a mobile application designed as an extension of an existing web application used as a school management ERP system. The app provides an easy-to-use interface for daily activities such as attendance, grading exams, and accessing the learning management system (LMS). The app also integrates mobile notifications via Firebase, which allows users to receive circulars, notices, and timeline posts. Additionally, the app has an efficient chat messaging structure that allows for live messaging and responses with images, voice, and documents. State management was implemented using Riverpod, and the app introduced different payment gateways for different clients for fee collection from parents.
